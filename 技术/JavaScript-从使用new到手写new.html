分析：

因为 new 的结果会返回一个对象，所以在模拟实现它的时候，我们也要建立一个新的对象，假设这个对象叫 obj。因为 obj 会具有构造函数（这里指Dog）里面的属性。我们使用 Dog.allpy(obj, arguments) 来给 obj 添加新的属性。

补充两点：

  1. __proto__ 和 constructor 属性是对象所独有的；
  2. prototype 属性是函数所独有的。
